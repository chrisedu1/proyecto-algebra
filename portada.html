Matrices

Las matrices son un conjunto bidimensional de números o símbolos
distribuidos de forma rectangular, en líneas verticales y horizontales,
de manera que sus elementos se organizan en filas y columnas.
Sirven para describir sistemas de ecuaciones lineales o diferenciales,
así como para representar una aplicación lineal.


Matriz inversa

La matriz inversa de una matriz es la matriz que,
al multiplicarse por la matriz original da,
como resultado la matriz identidad.

Multiplicación de Matrices

La multiplicación de matrices es una operación matemática fundamental
que combina dos matrices para producir una tercera matriz.
Esta operación se utiliza en una variedad de campos, incluyendo matemáticas,
física, informática, economía, ingeniería y muchos otros.
La multiplicación de matrices se rige por reglas específicas y puede no ser conmutativa,
lo que significa que el orden en el que se multiplican las matrices importa.
Para que las matrices A y B sean multiplicables, el número de columnas en A debe ser
igual al número de filas en B. Si A es una matriz de dimensiones m x n
(m filas y n columnas) y B es una matriz de dimensiones n x p,
entonces la matriz resultante C será de dimensiones m x p.

Sistemas de Ecuaciones

Un sistema de ecuaciones es un conjunto de ecuaciones algebraicas relacionadas que se utilizan para representar situaciones en las que varias incógnitas están sujetas a restricciones simultáneas. Cada ecuación en el sistema proporciona información sobre cómo las variables desconocidas se relacionan entre sí.

Método Gauss Jordan

El método de Gauss-Jordan es un algoritmo utilizado para
resolver sistemas de ecuaciones lineales y encontrar
las soluciones de las incógnitas. Este método es
una extensión del método de eliminación de Gauss
y es especialmente útil para encontrar la solución de
un sistema de ecuaciones en forma de matriz aumentada.

Regla de Cramer

La regla de Cramer se basa en la teoría de determinantes
y proporciona una forma sistemática de calcular las soluciones de
un sistema de ecuaciones utilizando determinantes y matrices auxiliares.
